MacOS 上安装配置 MySQL

发表于 2018-06-11 | 更新于 2018-07-10 | 阅读次数：

由于项目需要使用数据库来管理数据，所以最近开始学习 SQL。我打算使用的 DBMS 为MySQL，下面简单介绍如何在 MacOS 上安装使用 MySQL。

1. 下载 MySQL

在 MySQL 官网选择 MySQL Community Edition 版本，分别下载 MySQL Community Server 和 MySQL Workbench，如下图所示。

MySQL Workbench 是 MySQL 的一个交互工具，在学习 MySQL 时，强烈推荐使用它。
进入下载页面后会自动选择 MacOS 的平台，此时我下载的版本为 MySQL Community Server 8.0.11 和 # MySQL Workbench 6.3.10。
为了安装方便，以上两个都我们下载 .dmg 安装包。

2. 安装MySQL

分别打开两个安装包，直接安装即可。SQL 安装时需要为 root 账号设置一个密码，安装成功后选择启动 MySQL，在系统偏好设置可以看到 MySQL 选项。

SQL Workbench 直接拖拽到应用程序文件夹内，之后在 Launchpad 中会出现应用的图标。

3. 配置 MySQL

打开终端，在命令行中使用 SQL 命令来管理 MySQL 服务。如果返回 command not found，则需要检测服务是否启动。

服务启动后，MySQL 命令仍找不到，则需要配置环境变量。

4. 使用 MySQL Workbench

运行 MySQL Workbench，点击列出的 MySQL 数据库连接，如果没有的话，选择 New Connection。
但是新建后点击 Test Connection 按钮，出现报错。

Authentication plugin ‘caching_sha2_password’ cannot be loaded: dlopen(/usr/local/mysql/lib/plugin/caching_sha2_password.so, 2): image not found

Stackoverflow 上给出的解决方法是：

  1. 打开系统偏好设置 > MySQL > Initialze Database
  2. 输入新密码
  3. 选择 Use Legacy Password Encryption
  4. 重启 MySQL Server
  5. 重新连接数据库服务

好了！学习 MySQL 的前期工作已经准备好了，让我们开始吧！
