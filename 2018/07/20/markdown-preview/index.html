如何在 Sublime Text 3 预览 markdown 文档

发表于 2018-07-20 | 评论数： | 阅读次数：

介绍

在上一篇文章中，简单介绍了 Github 的使用。对于一个好的开源项目我们想把它 git clone 到本地来学习，其中有很多是 Markdown 格式的文档。那么在本地如何很好地编辑并预览呢？

你也许会想到专门在 Markdown 的软件中使用，比如 macOS 上的 Mou。但我们往往大多时候还是在一个类似 Sublime Text 3 的专业的编辑器中编辑，在编辑完成后预览。下面我们将介绍一个 Sublime Text 3 的插件 Markdown Preview，能够在浏览器中直接预览 Markdown 文件，并且支持 Github Flavored Markdown。

Markdown Preview 插件

1. 安装

组合按键 Ctrl(Command)+Shift+P 或是点击 Perference->Package Control 调出命令框，输入 Install，选择 Install Package。

在输入框中输入 Markdown Preview 可以看到插件，点击下载安装。因为我已经安装过了，所以搜索并没有显示。

2. 网页预览

通过组合按键 Ctrl+Shift+P 或点击 Preference->Package Control 调出命令面板，输入 mdp，选择 Preview in Browser。

选中后，将有两个选项：Github 和 Markdown。Github 则使用 Github 的 API 来解析，效果和在 Github 上预览一样。Markdown 则是普通 Markdown 显示，可能会与 Github 有差别。
下面我们看看使用 Github 选项预览的效果。

3. 目录设置

在任意位置插入[TOC] 即可生成目录。不过目录在 Github 选项中不显示，Markdown 选项则会显示目录。

4. 导出为 PDF

在浏览器预览后，在浏览器选择打印->另存为PDF即可。

总结

不用再下载单独的 Markdown 软件，我们只需要有写作利器 Sublime Text 3 就可以愉快地进行写作啦！
