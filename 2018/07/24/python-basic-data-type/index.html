Python 基本数据类型

发表于 2018-07-24 | 分类于 Python | 评论数： | 阅读次数：

现在正在夯实算法和数据结构的基础知识，之前一直采用 C++。但是 C++ 使用起来确实还是心有些力不从心，相比之下，Pyrhon 上起手来更快一些。以后也想从事和 Python 有关的工作，毕竟机器学习、数据挖掘现在正在浪潮，为何不去做一个弄潮儿呢？
之前也就是偶尔掉用 Python 的库，实现某个具体应用。如今要重新拾起 Python，先回顾一下 Python 的基本数据类型。

数值类型

  • int 整数
  • float 浮点数

布尔类型

  • bool 布尔型
  • 值：True and False
  • 操作符：and, or, and not

集合数据类型

有序集合

List 列表

list 用方括号括起，不同值之间用逗号隔开。
list 内的数据类型可以不同。
list 的操作如下表：

操作名 操作符 说明
索引 [] 获得序列元素值
连接 + 将两个序列连在一起
重复 * 重复连接多次
存在 in 元素是否存在序列中
长度 len 返回序列长度
提取 [:] 提取序列的部分元素

list 提供一系列方法帮助我们构建数据结构：

方法名 用法 说明
append alist.append(item) 将元素加到列表尾
insert alist.insert(i, item) 将元素插入到列表的第 i 个位置
pop alist.pop() 删除并返回列表最后一个元素
pop alist.pop(i) 删除并返回列表第 i 个元素
sort alist.sort() 将列表排序
reverse alist.reverse() 将列表逆转
del del alist[i] 删除第i个元素
index alist.index(item) 返回元素第一次出现的序号
count alist.count(item) 返回元素的出现次数
remove alist.remove(item) 删除第一次出现的元素

函数 range 常常与 list 一起使用。range 产生一个 range 对象，表示值的序列。对其使用 list 函数，能将其转化为列表显示。
range 对象表示了一系列整数，默认从0开始，也可以自定义范围。输入第三个参数可以作为步长，如我们倒序取从10开始的数：

String 字符串

字符串是字符的序列集合。因为字符是序列，所以之前的操作都可以进行。除此之外，还有字符串自己的方法。

方法名 用法 说明
center astring.center(w) 将字符串在宽度 w 的范围内居中
count astring.count(item) 统计 item 在字符串中出现的次数
ljust astring.ljust(w) 返回宽度 w 的左对齐字符串
rjust astring.rjust(w) 返回宽度 w 的右对齐字符串
lower astring.lower() 返回小写字符串
upper astring.upper() 返回大写字符串
find astring.find(item) 返回 item 第一次出现的序号
split astring.split(schar) 以 schar 来分割字符串

列表和字符串的主要区别在于：列表可以更改，而字符串不行。

Tuples 元组

元组同列表一样也可以包含不同数据类型，但元组不可改变。元组内元素用逗号分隔，用圆括号括起。作为序列，之前的操作同样适用。

无序集合

Set 集合

set 是无序数据对象集合，并且不可更改。set 内元素用逗号分隔，并用大括号括起。一个空的集合用 set() 表示。
集合的操作：

操作名 操作符 说明
成员资格 in 是否属于集合
长度 len 集合成员个数
| aset | otherset 并集
& aset & otherset 交集
- aset - otherset 差集
<= aset <= otherset 判断子集

集合的方法（其中前四种有对应操作）：

方法名 用法 说明
union aset.union(otherset) 并集
intersection aset.inersection(otherset) 交集
difference aset.difference(otherset) 差集
issubset aset.issubset(otherset) 判断子集
add aset.add(item) 添加
remove aset.remove(item) 删除
pop aset.pop() 删除任意元素
clear aset.clear() 删除所有元素

Dictionary 字典

字典是键值对（key-value）的集合。键值对记为：“key:value”。每个键值对用逗号隔开，大括号括起。
字典的操作符如下：

操作符 用法 说明
[] myDicdt[k] 返回键 k 的值，不存在则返回错误
in key in adict 判断键是否存在
del del adict[key] 删除键记录

字典的方法如下：

方法名 用法 说明
keys adict.keys() 返回 dict_key 对象的字典键名
values adict.values() 返回 dict_values 对象的字典值
items adict.items() 返回 dict_items 对象的键值对
get adict.get(k) 返回键 k 对应的值，不存在则返回 None
get adict.get(k, alt) 返回键 k 对应的值，不存在则返回 alt

keys, values 和 items 都返回包含感兴趣值的对象，可以使用 list 函数将其转化为列表。
