Python读取mat文件

发表于 2019-02-28 | 分类于 Python | 评论数： | 阅读次数：

在实验中我们常常将数据保存为Matlab的mat文件，但有时需要用Python来处理数据，scipy库可以很好地支持mat文件的读写。

使用loadmat()函数读取mat文件，得到字典格式的数据。除了mat保存的矩阵数据外，还包括一些附加信息。我们以变量名为关键字就可以获得数据内容，可以看到这个mat文件中保存了一个大小为(2151, 6)的矩阵。
