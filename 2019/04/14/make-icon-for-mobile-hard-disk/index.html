在Mac上制作移动硬盘图标

发表于 2019-04-14 | 分类于 杂项 | 评论数： | 阅读次数：

随着数据量的不断增加，移动硬盘成为我们办公生活不可或缺的工具。使用MacOS的小伙伴会发现，连接硬盘到电脑之后在桌面会出现一个黄色的硬盘图标。这些图标都是一个样，显得平淡无奇。如果想个性化显示每个硬盘那么该怎么样呢？

我们可以将硬盘图标换成硬盘的图片，下面一步一步来实现吧！

  1. 下载硬盘商品图片。比如在京东商城找到对应的商品，在左下角中选择一张想要作为图标的图片，点击“查看图片”。此时浏览器会打开新的页面并显示图片，但是图片尺寸相对较小，我们需要查看大图。更改图片地址中54x54为600x600则显示大图，保存图片即可。
  2. 下载图标制作软件Image2icon。在App Store中下载Image2icon，这是一款专门制作图标的软件。使用起来非常方便，只需要把图片拖入然后就能转换成我们想要的格式。
  3. 制作硬盘图标。将我们选好的图片拖入软件，选择Apple的icons格式图标导出即可。
  4. 替换默认硬盘图标。在桌面选中硬盘，按CMD+i显示硬盘信息。将导出的图标拖入替换原始图标即可。

看看我们的硬盘图标显示效果吧！
